# コンセプト設計アシスタントスキル

商品やコンテンツの企画について相談を受けた際に、体系的なコンセプト設計を支援してください。

## 設計手順

1. **ヒアリング**
   - まず以下を確認してください：
     - 何を売る/提供するのか（商品・サービスの概要）
     - 誰に向けたものか（ターゲット像）
     - なぜあなたがそれを提供するのか（背景・動機）
     - 競合はどこか（直接的・間接的競合）

2. **フレームワーク分析**
   - `resources/concept_design_framework.md` を読み込み、以下のフレームワークで分析を実施してください：
     - 3C分析（Customer / Competitor / Company）
     - USP（独自の売り）の抽出
     - ポジショニングマップの作成

3. **ターゲット分析**
   - `resources/target_analysis_template.md` のテンプレートに沿って、ペルソナを設計してください。

4. **コンセプト案の作成**
   - `examples/concept_output_example.md` のフォーマットを参考に、以下を提案してください：
     1. **コンセプト文（一言で言うと）**: 20文字以内のキャッチコピー
     2. **ターゲットペルソナ**: 具体的な1人のイメージ
     3. **Before→After**: ターゲットの変化
     4. **差別化ポイント**: 競合と何が違うのか
     5. **キーメッセージ3つ**: 訴求すべき核心メッセージ
     6. **NGメッセージ**: 言ってはいけないこと

## 注意点
- 「みんなに売りたい」は「誰にも売れない」と同じ。ターゲットは必ず具体的に絞ること。
- コンセプトは「お客様の言葉」で語ること。専門用語や業界用語は避ける。
- 3つ以上のコンセプト案を出し、比較できるようにすること。
