# コンテンツ品質チェッカースキル

提示された文章に対して品質チェックを行い、改善点を指摘してください。

## チェック手順

1. **自動スキャンの実行**
   - 対象がファイルの場合、以下のスクリプトを実行してください。
   - コマンド: `python scripts/quality_checker.py <ファイルパス>`
   - チャットに直接貼り付けられた場合は、目視で確認してください。

2. **品質チェックリストの確認**
   - `resources/quality_checklist.md` を読み込み、各項目についてコンテンツを評価してください。

3. **レポートの作成**
   - `examples/quality_review_example.md` のフォーマットを参考に、以下の構成で回答してください：
     1. **品質スコア**: 各観点の5段階評価
     2. **自動検出された問題**: スクリプトで検出した項目
     3. **読みやすさの改善**: 文長・文末・段落の改善案
     4. **表現の改善**: より伝わる言い回しの提案
     5. **修正済みテキスト**: 改善箇所を反映した修正版（任意）

## 注意点
- 文体は「正しさ」ではなく「効果」で評価してください。口語体が効果的なら口語でOK。
- Note記事とX投稿では基準が異なります。X投稿は短さ・インパクト重視で評価してください。
- 誤字脱字の指摘だけでなく、「なぜ変えるべきか」の理由も添えてください。
