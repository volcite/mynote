# マーケティングコピーレビューアースキル

提示されたセールスコピー・LP・広告文に対して、マーケティング心理学の観点からレビューを行い、改善点を提案してください。

## レビュー手順

1. **自動スキャンの実行**
   - 対象がファイルの場合、以下のスクリプトを実行して基本データを取得してください。
   - コマンド: `python scripts/copy_checker.py <ファイルパス>`
   - チャットに直接貼り付けられた場合は、目視で分析してください。

2. **心理テクニック診断**
   - `resources/psychology_checklist.md` を読み込み、各心理テクニックが適切に使われているかチェックしてください。
   - 不足している心理テクニックを特定し、追加提案してください。

3. **セールスライティング知識との照合**
   - `resources/sales_writing_knowledge.md` のエッセンスを参照し、コピーの改善点を洗い出してください。

4. **レビューレポートの作成**
   - `examples/review_output_example.md` のフォーマットを参考に、以下の構成で回答してください：
     1. **全体評価**: 5段階スコアと総評
     2. **心理テクニック使用状況**: 各テクニックの使用有無と効果度
     3. **パワーワード分析**: 効果的な表現と改善余地
     4. **CTA分析**: 行動喚起の数・配置・効果
     5. **改善提案**: 具体的な修正案（Before/After形式）
     6. **最優先アクション**: 今すぐ改善すべき1点

## 注意点
- 「ただ心理テクニックを詰め込め」ではなく、ターゲット読者に合った使い方を提案してください。
- 改善案は必ず「具体的な文章」で示してください。「もっと強くしましょう」のような抽象的な指摘はNG。
- 誇大表現や景品表示法に抵触しうる表現がないかも確認してください。
