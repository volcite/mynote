# 公式LINE垢バン＆メルマガ送信エラー完全対策スキル

このスキルは、ユーザーが作成したコンテンツ（LINE配信、メルマガ、LP、ブログ記事など）や、計画している運用フローに対して、**LINE公式アカウントの凍結（BAN）リスク**および**メールの不達（スパム判定）リスク**を診断し、具体的な改善案を提示するために使用します。

## 診断手順

1. **自動スキャンの実行（コンテンツの場合）**
   - 対象がファイルの場合、以下のスクリプトを実行してNGワード等の基本チェックを行ってください。
   - コマンド: `python scripts/ban_risk_analyzer.py <ファイルパス>`
   - チャットに直接貼り付けられた場合や運用フロー・構造の相談の場合は、目視で分析してください。

2. **リスクパターンとの照合**
   - `resources/ban_risk_patterns.md`（リスクの仕組みと具体例）および `resources/official_guidelines.md`（公式ガイドライン・法的要件）を参照し、対象コンテンツ・運用フローのリスク（コンテンツ、行動、リスト品質、構造）を判定してください。
   - 判断に迷う場合は `resources/checklists.md` の判断フローチャートとチェックリストを活用してください。

3. **診断レポートと改善提案の作成**
   - `examples/feedback_example.md` のフォーマットや `resources/ban_risk_patterns.md` (安全な言い換え例) を参考に、以下の構成で回答してください：
     1. **危険度総合判定**: （低/中/高/即死レベル）
     2. **検出されたリスク要因**: 具体的なNG表現や危険なフローの指摘
     3. **改善提案**: 言い換え案（ピリオド挿入テクニック含む）や運用フローの修正案
     4. **今後の運用アドバイス**: BANされた際の影響を小さくする構造化提案など

## 注意点
- 凍結は事前警告なく突然停止されるため、少しでも「詐欺・スパム業者と同じパターン」とみなされうる表現は厳しく指摘してください。
- 出力時には必ず「具体的な改善案（言い換えや運用変更）」を含めてください。
