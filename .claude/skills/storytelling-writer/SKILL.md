# ストーリーテリングスキル

文章にストーリーテリングの要素を組み込み、読者の記憶に残る文章を作成してください。

## 使い方

1. **物語構造の選定**
   - `resources/storytelling_framework.md` を読み込み、コンテンツの目的に合った物語構造を選んでください。

2. **感情トリガーの設計**
   - `resources/emotion_triggers.md` を参照し、適切な感情トリガーを選んで配置してください。

3. **ストーリーの構築**
   - `examples/storytelling_samples.md` の実例を参考に、以下の要素を含めてください：
     1. **共感できる主人公**: 読者が「自分と同じだ」と感じる人物
     2. **具体的な障壁**: 乗り越えるべき課題や困難
     3. **感情の起伏**: 喜び・恐怖・怒り・驚き・共感の波
     4. **転機（ターニングポイント）**: 物語が動く瞬間
     5. **教訓・メッセージ**: 読者に持ち帰ってほしい一つのメッセージ

## 適用場面
- **自己紹介記事**: 自分の人生をストーリーとして語る
- **セールスコピー**: 商品の誕生背景やお客様の変化をストーリーで描く
- **X投稿**: ミニストーリーで読者の手を止める
- **ウェビナー台本**: 冒頭のフックやケーススタディにストーリーを活用

## 注意点
- ストーリーは「嘘」ではなく「事実の編集」。事実に基づいて、最も伝わる順番で語る。
- 1つのストーリーには1つのメッセージ。詰め込みすぎない。
- 五感（視覚・聴覚・触覚・嗅覚・味覚）の描写を入れると臨場感が増す。
