# ストーリーテリング フレームワーク集

## 1. ヒーローズジャーニー（英雄の旅）

最も強力な物語構造。読者を「主人公の旅」に巻き込む。

| ステージ | 内容 | 例（こまの自己紹介） |
|---|---|---|
| 日常の世界 | 主人公の普段の生活 | 長野育ちの大人しい少年時代 |
| 冒険への召命 | 変化のきっかけ | パラグアイ・ベルギーへの挑戦 |
| 試練 | 困難・失敗 | 詐欺で350万の借金、ホームレス |
| 師との出会い | 導き手との出会い | 「どうなりたいの？」と聞いてくれた人 |
| 変容 | 主人公の変化 | AIとの出会い、業務効率化への転身 |
| 帰還 | 得た知恵を持ち帰る | 「目の前の人一人ひとりと向き合う」 |

## 2. Before → After → Bridge（BAB）

最もシンプルで汎用性が高い。セールスコピーに最適。

```
Before: 読者の現在の辛い状況を描写する
    ↓
After:  理想の未来を具体的にイメージさせる
    ↓
Bridge: その橋渡しとなる手段（商品・サービス）を提示する
```

**ポイント**: Before と After のコントラストが大きいほど効果的。

---

## 3. 問題 → 葛藤 → 解決（PCS）

短いストーリーに最適。Xポストやメルマガの冒頭に使える。

```
Problem:  一つの具体的な問題を提示
    ↓
Conflict: その問題で苦しんだ具体的な場面
    ↓
Solution: 解決策との出会い・結果
```

---

## 4. 3幕構成

映画・ドラマと同じ構造。長い記事やウェビナーに使える。

| 幕 | 比率 | 内容 |
|---|---|---|
| 第1幕：設定 | 25% | 世界観の提示・主人公の紹介・問題の提示 |
| 第2幕：対立 | 50% | 試練・挫折・葛藤・仲間との出会い |
| 第3幕：解決 | 25% | クライマックス・解決・教訓 |

---

## 5. ストーリーアーク（感情曲線）

読者の感情を意図的に上下させる設計。

```
感情 ↑
     |        ★転換点
     |       / \
     |      /   \     ★解決
     |     /     \   /
     |    /       \ /
     |   /
     |  /  ★問題発生
     | /
     |/ ★日常
     +------------------------→ 時間
```

**ルール**:
- 上がったら必ず下げる、下がったら必ず上げる
- クライマックスは全体の75%地点に置く
- 最後は「日常」より少し上で終わる（希望を残す）

---

## ストーリーの5つの要素チェックリスト

- [ ] **主人公**: 読者が感情移入できる人物がいるか
- [ ] **欲求**: 主人公は何を求めているか明確か
- [ ] **障壁**: 目標を阻む具体的な障害があるか
- [ ] **変化**: 主人公は物語を通じて変わるか
- [ ] **教訓**: 読者が持ち帰る「一言」は何か
