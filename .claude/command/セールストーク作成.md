# セールストーク作成ワークフロー

このワークフローは、ユーザーのビジネスに基づき「1時間の商談で売れるセールストーク台本」を作成します。

## 使用方法

1. Phase 0: 事前準備（リサーチ・仮説構築・種まきメッセージ）
2. Phase 1: ヒアリング（商品・ターゲット・ストーリーの素材集め）
3. Phase 2: コンセプト設計（FPCSループと反論処理の設計）
4. Phase 3: 構成案の作成（1時間を5パートに分割）
5. Phase 4: セールストーク台本の執筆
6. Phase 5: ファイル保存＆フォロー台本

# ★★★ 知識ベース（Reference Knowledge） ★★★

## 1. ★大原則★ Sales is Help（セールスは人助け）

- **セールスは人助け**: 必要な人に必要なものを届ける「助け」である
- **顧客を全肯定**: 絶対に否定しない。「あなたは悪くない」と言い続ける
- **To Who（誰に向けて）**: 目の前の「助けたい一人」に集中する

## 2. ★最重要★ FPCS 4ステップ（ループで回す）

```
1. Future（未来）── ワクワクする究極の未来を提示
2. Problem（問題）── 気づいていない問題を指摘（否定ではない）
3. Cause（原因）── 第3者のせい。「あなたは悪くない」
4. Solution（解決策）── ここで初めて商品を提示
```

**★ポイント★**: 未来→問題→原因→解決策 をグルグル回す（感情の波）。商談では最低3回ループ。

## 3. ★NEW★ 診断者ポジション — 売る側から「処方する側」へ

- **医者のポジション**を取る。「紹介します」ではなく「診させていただきます」
- 医者が「この薬を買ってください」とは言わない → 「あなたにはこれが必要です」
- 商品提案は「処方箋」→ 相手が「お願いします」と言う構造を作る
- **キーフレーズ**: 「今日は〇〇さんの状況を診させていただいて、本当に必要なものがあればお伝えします」

## 4. ★NEW★ アイデンティティ・シフト — 行動ではなく「存在」を変える

- 人は「行動を変えろ」と言われても変われない。しかし「あなたは本来こういう人だ」と言われると行動が勝手に変わる
- **売るのは商品ではなく「本来のあなた」に戻る切符**

```
旧: 「この講座を受ければ月収100万になります」（行動→結果）
新: 「〇〇さんは本来、月収100万を稼ぐ力を持ってる人なんです。
     ただ、正しいやり方を知らないだけ。
     もし今の能力に正しい方向性が加わったら——どうなると思います？」
     （存在の肯定→可能性の自己発見）
```

## 5. 商品理解は「中身」ではなく「側（がわ）」

- **中身**（カリキュラム・機能）→ 売れない
- **側**（どんな問題に効くか・どう変わるか）→ これだけ伝える
- 顧客に理解させるべきは: ①現状のヤバさ ②解決手段（商品）

## 6. ラポール形成（診断者として）

- **目的を先に伝える**: 「診させていただきます」と宣言してからヒアリング
- **趣味の話はNG**: 目的に沿ったヒアリングのみ
- **自己紹介は商品に寄せる**: 実績をコンテンツに関連付けて語る
- **聞く80%**: 質問で深掘りし、相手に語らせる
- **決断パターンを読む**: 即決型かじっくり型かを早期に判断

## 7. テストクロージング（各パートで承認を取る）

- **小さな「はい」を積み重ねる**: 最終クロージングを自然な流れにする
- **二者択一質問**: 「AとBどちらが気になりますか？」
- **仮定質問**: 「もし結果が保証されるとしたら、やりたいですか？」
- **★「やるとしたら」質問★**: 「やるかやらないか」ではなく、「やるとしたら」の前提で聞く。例: 「もしこの方向で進めるとしたら、最初の2ヶ月、1日何時間くらい確保できそうですか？」→ 相手の頭が勝手に「やる前提」で動き始める

## 8. 3つの反論処理（3 Beliefs）

| 壁 | 顧客の心の声 | 対策 |
|----|-------------|------|
| **Vehicle（方法）** | 「うまくいくの？」 | 社会的証明を見せる |
| **Internal（自分）** | 「私にもできる？」 | 似た境遇の成功事例 |
| **External（環境）** | 「時間/お金がない」 | リフレーミング |

## 9. Feel-Felt-Found法（想定外の反論への対応）

```
Step 1 - Feel: 「その気持ち、よく分かります」（受け止め）
Step 2 - Felt: 「〇〇さんも最初は同じことを言ってました」（共感）
Step 3 - Found: 「でも実際やってみたら〇〇だったんです」（発見）
```

**反論のオウム返し**: 「『時間がない』ということですね？じゃあ時間の問題が解決したら、やりたいですか？」→ 本当の反論を切り分ける

## 10. バリュースタック＆価格演出

- 構成要素を分解し**それぞれに市場価格**をつける → 合計額 → 「でも今回は〇〇円」
- **★5倍ルール★**: バリュースタック合計額は**販売価格の最低3倍、理想は5倍以上**に設計する。合計額が販売価格を下回ると「損してる」計算になり逆効果
- **アンカリング**: 業界相場を先に出す
- **★価格推測テクニック★**: 価格提示の前に「ここまでの内容、いくらくらいだと思いますか？」と相手に推測させる → 相手の回答がアンカーとなり、バリュースタック合計額との差が際立つ
- **日割り計算**: 「1日〇〇円。コーヒー1杯分」
- **投資vs消費**: 回収期間は**現実的な数字**を使う（月商10万の人に「1ヶ月で回収」は嘘臭い → 「6ヶ月で回収」の方が信頼される）
- **★ROIシミュレーション★**: その場で相手の数字を使って投資対効果を計算して見せる。「〇〇さんの場合、月商が今の3倍になれば、差額×6ヶ月で回収。投資倍率は〇倍です」
- **★リスクリバーサル★**: 相手のリスクを取り除く保証を提示する。例: 「3ヶ月で成果ゼロなら追加3ヶ月無料延長」「最初の1ヶ月で合わなければ全額返金」。高単価ほど必須
- **★決済手段の提示★**: 「一括と分割、どちらがいいですか？」→ 「買うか買わないか」から「一括か分割か」へ論点をシフトさせる。高単価商品では分割の選択肢を先に出す
- **価格提示後は沈黙**: 焦って値引きしない

## 11. 購入の3条件

1. **結果を出せる確信**: 「自分でもできる」と思わせる
2. **明確なゴール**: 具体的にイメージできる
3. **ベネフィット ＞ 価格**: 得られるものが支払いを上回る

## 12. 共通の敵（感情のぶつけ先）

- 感情をぶつける先は常に**第3者**（業界・常識・教育）
- 相手には絶対にぶつけない
- ダークヒーロー的立ち位置：「俺がぶっ壊してやる」→ 応援したくなる

## 13. ハートフルストーリー

- 通常: 最終クロージング前（第5パート）に入れ、感情を揺さぶる
- **★高単価（100万超）の場合★**: 第3パートのV字ストーリーと一体化させる。感情が最高潮の状態で処方箋（第4パート）に入ることで、論理で正当化する流れが強くなる。**高単価は感情→論理の順番が鉄則**
- 第三者の実話（家族・クライアント）→ なぜこの商品を提供するか
- 共感を生む感情は「怒り・悲しみ・悔し涙」

## 14. 希少性と緊急性

- **なぜ今か**: 論理的に説明
- **なぜ限定か**: 物理的制限を明示
- **嘘はNG**: 事実ベースでなければ信頼を失う

## 15. カリスマテンプレート（常識破壊）

- **「え？」→「なるほど」**: 常識を壊し、新しい価値観を植え付ける
- ノウハウを教えるのではなく「気づきを与える」

## 16. レトリック質問（先回り処理）

- 「今、『自分には関係ない』と思いましたよね？」→ 相手の心を読んでいる感を出す
- 商談中に**最低3回**

## 17. 痛みの増幅質問（3段階）

```
Level 1（事実）: 「今の売上って月いくらですか？」
Level 2（感情）: 「それって、正直どんな気持ちですか？」
Level 3（恐怖）: 「このまま1年変えなかったら、どうなると思います？」
```

**★ゴールデンタイム★**: 相手が「このままじゃマズい」と言った瞬間 → Solutionが刺さる

## 18. 温度感チェック＆ルート分岐

```
【🔥 熱い】→ スピードアップ。教育を短縮し早めに提案へ
【😐 普通】→ 通常ルート。質問を増やして巻き込む
【❄️ 冷たい】→ 止める。「ぶっちゃけどうですか？」と本音を聞く
　　　　　　 → それでも冷たいなら無理に売らない。関係維持に切り替え
```

## 19. 戦略的沈黙（間の支配）

| タイミング | 秒数 | 効果 |
|---|---|---|
| 痛みを感じた後 | 2〜3秒 | 痛みが深く染みる |
| 常識破壊の直後 | 1〜2秒 | インパクト増幅 |
| 価格提示の直後 | 3〜5秒 | 消化させる |
| ストーリー山場の後 | 2〜3秒 | 感情の余韻 |
| 最終クロージング後 | ∞ | 先に喋ったら負け |

### ★沈黙の「質」★

最終クロージング後の沈黙は、ただ黙るのではない：
1. 穏やかな笑顔を保つ（圧をかけない）
2. 相手の目を見る（逸らさない、睨まない）
3. 微かにうなずく（「あなたは正しい決断ができる」を体で伝える）
4. 呼吸をゆっくり（場を安定させる）
→ 相手は沈黙の中で自分自身と対話している。あなたの仕事は「安全な空間を作ること」

## 20. ★NEW★ エネルギー設計 — 言葉ではなく「場」で売る

```
★エネルギーマップ（各パートの空気感）★

第1パート: 【軽い・温かい】 友達の家に遊びに来た感覚。声は明るく、テンポ良く
第2パート: 【真剣・深い】  医者が患者に向き合う感覚。声は落ち着いて、ゆっくり
第3パート: 【熱い・情熱的】 可能性に火をつける感覚。声に力を込めて、前のめりに
第4パート: 【冷静・プロ】  投資家に提案する感覚。論理的に、数字で語る
第5パート: 【静か・確信】  「あなたなら大丈夫」という安心感。穏やかに、深く
```

## 21. ★NEW★ 未来体験（Guided Visualization）— 想像ではなく「体感」

Futureパートで未来を「語る」だけでなく、**その場で体感させる**。

```
「ちょっとイメージしてみてください。
 朝起きて、スマホを開いたら、寝てる間に3件の問い合わせが来てる。
 しかも全部、あなたの理想のお客さん。
 コーヒー飲みながら、余裕を持って返信する。
 午前中には1件の成約が決まる。
 ……そんな毎日、嫌ですか？」

→ 相手が笑ったら勝ち。「え、それ最高じゃないですか」と言ったらもう売れている。
```

## 22. ★NEW★ ミラーリング＆ペーシング — 相手の言葉で売る

```
★相手の言葉をキャッチし、そのまま使い返す★

相手「なんか、ずっとモヤモヤしてて…」
→ 「そのモヤモヤ、よく分かります」
→ 「そのモヤモヤが晴れる方法があるんです」
→ 「モヤモヤから卒業しませんか？」

※こちらの言葉で翻訳しない。「不安」「焦り」に言い換えない。
　相手の言葉そのものが最強のセールスコピー。
```

# ★★★ AIがやりがちな失敗と対策（Absolute Rules） ★★★

## 失敗1: 一方的なプレゼンになる（聞かない）
- **対策**: ★聞く8割・話す2割。各パートに質問を入れる★

## 失敗2: 問題提起が「否定」になる
- **対策**: ★第3者のせいにする。「あなたは悪くない」を必ず入れる★

## 失敗3: 商品の「中身」を語りすぎる
- **対策**: ★「側（がわ）」で語る。どう変わるかだけ伝える★

## 失敗4: クロージングが唐突
- **対策**: ★テストクロージングを積み重ね、最終は「確認」にする★

## 失敗5: ネガティブの使い方を間違える
- **対策**: ★「全員に向いてるわけじゃない」で正直さと信頼を見せる★

## 失敗6: 反論に対して「論破」しようとする
- **対策**: ★Feel-Felt-Found法。反論は「助けを求めるサイン」★

## 失敗7: 自分の話が多すぎる（主役がズレる）
- **対策**: ★V字ストーリーは「あなたの話を聞いて思い出した」から入る。主語を常に相手に戻す★
- 「私のストーリー聞いてください」はNG → 「〇〇さんと同じ状況だった時の話なんですけど」

## 失敗8: エネルギーが一定で変化がない
- **対策**: ★エネルギーマップに沿って空気感を変える。抑揚のない商談は眠くなる★

## 禁止表現

- **AI特有**: 「効率化」「最適化」「包括的な」「網羅的な」「体系的な」
- **弱気**: 「〜かもしれません」「〜だと思います」
- **上から目線**: 説教臭い語り口
- **直接否定**: 「間違っている」「ダメです」

# 実行プロセス

## Phase 0: 事前準備（商談前リサーチ）

```markdown
## 事前リサーチシート

1. **相手の情報**: 氏名・年齢層・職業・SNS・よく使う言葉・過去の商材購入歴
2. **相手のステージ**: 完全初心者/初心者/中級者/上級者
3. **商談の経緯**: 流入元・事前に見た情報・相手からか自分からか
4. **購買パターンの推測**: 即決型？じっくり型？比較型？
```

### 事前準備で生成するもの

- **仮説リスト**: 相手の問題・欲しい未来の仮説3つ
- **キーワードリスト**: 反応しそうな言葉3〜5個
- **種まきメッセージ**: 商談前日のLINE/メール（期待値UP）
- **アイデンティティ仮説**: 「この人が本来持っている力」は何か

```
★種まきメッセージの例★
「明日はお時間いただきありがとうございます。
〇〇さんの状況を事前に少し拝見させていただいたんですが、
正直、めちゃくちゃ可能性を感じました。
明日の面談で、その理由を詳しくお伝えしますね。」
```

## Phase 1: ヒアリング

```markdown
## セールストーク作成ヒアリングシート

1. **商品/サービス**: 何を売るか？価格帯は？
2. **ターゲットの現状**: 「良かれと思って」何を頑張っているか？
3. **究極の未来（Future）**: 人生が変わるレベルの未来は？
4. **気づいていない問題（Problem）**: 信じている常識で実は間違っているものは？
5. **第3者の敵（Cause）**: 誰（何）のせいにするか？
6. **V字ストーリー＆社会的証明**: 失敗→成功ストーリー＋他の成功事例
7. **3つの反論**: Vehicle/Internal/Externalで想定される不安
8. **バリュースタック**: 商品に含まれる要素を全てリスト
9. **緊急性と希少性**: なぜ「今」決断しないといけないか？
10. **ハートフルストーリー**: 最後の感情プッシュに使えるストーリー
11. **商談の形式**: 対面/オンライン、事前情報、1対1/1対多
12. **過去の断り文句**: よく言われる反論や断り文句
13. **ターゲットの本来の力**: この人が持っている素質・才能・経験で活かせるものは何か？
```

## Phase 3: 構成案の作成

### 1時間商談構成テンプレート

#### 第1パート：アイスブレイク＆診断宣言【0〜5分】

**【心理ゴール】**: 「この人は信頼できそう。話を聞いてみよう」
**【エネルギー】**: 軽い・温かい ── 友達の家に来た感覚

*   **質問から入る**: 「今日はどんな期待を持って来てくださいましたか？」
*   **決断パターンを読む**: 「大きな買い物する時って、即決派？じっくり派？」
*   **過去の投資経験**: 「自己投資とか、セミナー・講座って受けたことあります？」
*   **診断者宣言**: 「今日は〇〇さんの状況を診させていただいて、本当に必要なものがあればお伝えします。合わなければ正直にそう言います」
*   **テストクロージング**: 「もし本当に必要なものだったら、前向きに検討いただけますか？」

**【温度チェック】**: 回答の熱量で初期温度を判定
**【ミラーリング開始】**: 相手が使った言葉をメモし始める

---

#### 第2パート：ヒアリング＆痛みの顕在化【5〜15分】── FPCS 1巡目

**【心理ゴール】**: 「やばい、このままだとマズいかも…でも、この人は分かってくれてる」
**【エネルギー】**: 真剣・深い ── 医者が患者に向き合う感覚

*   **Future 1（未来体験）**: 「ちょっとイメージしてみてください。〇〇な朝…（Guided Visualization）」
*   **Problem 1**: 「今それが実現できていない原因って何だと思いますか？」→ 相手に語らせる
*   **★痛みの増幅質問★**: Level 1→2→3 + ★沈黙★（2〜3秒）
*   **Cause 1（第三者のせい）**: 「実はそれ、〇〇さんのせいじゃないんです」
*   **★アイデンティティ・シフト（伏線）★**: 「でも〇〇さん、ここに来てるってことは、もう答えに近い場所にいるってことだと思います」
*   **テストクロージング**: 「今の話、なるほどと思いましたか？」
*   **★ミラーリング★**: 相手が使った痛みの言葉をキャッチしてメモ

**【温度チェック】**: 🔥 → 教育短縮 ｜ 😐 → 通常 ｜ ❄️ → 本音を聞く

---

#### 第3パート：教育＆気づき＆アイデンティティ・シフト【15〜30分】── FPCS 2巡目

**【心理ゴール】**: 「自分は本来やれる人間だ。この方法なら自分でもいける」
**【エネルギー】**: 熱い・情熱的 ── 可能性に火をつける感覚

*   **カリスマテンプレート**: 「え？」→ ★沈黙1〜2秒★ →「なるほど」
*   **Future 2**: 「実はこういう方法で、こんなことが可能になるんです」
*   **Problem 2**: 「でも多くの人が失敗する理由があって…」
*   **Cause 2（共通の敵）**: 「これは〇〇業界の構造的な問題で…」→ 感情をぶつける
*   **Solution 2（チラ見せ）**: 「この問題を根本から解決する方法があるんです」
*   **V字ストーリー（相手主役で入る）**: 「〇〇さんが言ってる状況、手に取るように分かります。実は私もまったく同じところにいたから——」
    *   ★沈黙★ ストーリー山場の後、余韻を残す
*   **★高単価の場合：ハートフルストーリーをここに挿入★**: 100万超の商品の場合、V字ストーリーの直後にハートフルストーリー（なぜこの商品を提供するのか）を入れる。感情が最高潮の状態で第4パート（処方箋）に入ると、論理で正当化する流れが強くなる
*   **社会的証明**: 「〇〇さんと同じ状況だったAさんも結果を出しました」
*   **★アイデンティティ・シフト（本番）★**:
    「〇〇さん、正直に言わせてください。
    　〇〇さんは本来、〇〇できる力を持ってる人だと思います。
    　ただ、正しいやり方を知らなかっただけ。
    　もし今の〇〇さんの[強み]に、正しい方向性が加わったら——どうなると思いますか？」
    ★質問★ 相手に未来を自分の口で語らせる
*   **レトリック質問**: 「今『無理かも』って思いましたよね？」→ 潰す
*   **テストクロージング**: 「この方法に可能性を感じますか？」

**【温度チェック】**: 🔥「具体的に聞きたい」→ 即第4パートへ ｜ ❄️ → 事例追加or本音を聞く

---

#### 第4パート：処方箋＆バリュースタック【30〜45分】── FPCS 3巡目

**【心理ゴール】**: 「これは投資する価値がある。始めたい」
**【エネルギー】**: 冷静・プロ ── 投資家に提案する感覚

*   **The Pivot**: 「方向性は分かった。でも一人でやると〇〇の壁にぶつかります」
*   **処方箋の提示**: 「〇〇さんの症状に対する処方箋をお渡しします」（★診断者ポジション★）
*   **商品は「側」で語る**: 「これを使うと〇〇が解決して〇〇な未来が手に入ります」
*   **バリュースタック**: 1つずつ紹介 → 合計額（★合計額は販売価格の3〜5倍以上★）
*   **アンカリング**: 業界相場 → 自社価格
*   **★ROIシミュレーション★**: 相手の実数字を使って投資対効果をその場で計算。「〇〇さんの場合…」
*   **★価格推測★**: 価格提示の前に「ここまでの内容、いくらくらいだと思いますか？」と聞く → 相手の回答がアンカーになる
*   **価格提示**: ★沈黙★ 3〜5秒。焦って値引きしない
*   **投資回収スキーム**: 現実的な期間で語る（ターゲットの現在月商から逆算）
*   **日割り計算**: 「1日〇〇円」
*   **★リスクリバーサル★**: 「〇〇保証があるので、リスクはほぼゼロです」
*   **★決済手段の提示★**: 「一括と分割、どちらがいいですか？」→ 論点を「買う/買わない」から「一括/分割」へシフト
*   **反論処理（3 Beliefs）**: 先回りで潰す
*   **★ミラーリング★**: 相手が使った言葉で商品メリットを語る
*   **ネガティブ活用**: 「全員向けじゃない。〇〇な人には合わない」→ 信頼＆自己選択
*   **テストクロージング（「やるとしたら」形式）**: 「もしこの方向で進めるとしたら、〇〇はどうしますか？」→ やる前提で考えさせる

**【温度チェック】**: 🔥「やりたい」→ 即クロージング ｜ 😐「いいけど…」→ 引っかかりを聞く ｜ ❄️ → Feel-Felt-Found

---

#### 第5パート：決断＆限定性【45〜60分】

**【心理ゴール】**: 「今、変わろう」
**【エネルギー】**: 静か・確信 ── 「あなたなら大丈夫」という安心感

*   **ハートフルストーリー**: なぜこの商品を提供するのか。理念・ミッション（★高単価は第3パートで済んでいる場合、ここでは短い要約のみ。通常単価はここでフルで語る★）
    ★沈黙★ 感情の余韻
*   **要約**: 「まとめると、今〇〇の状態から〇〇の未来へ行く手段がこれです」
*   **希少性と緊急性**: 「この条件は〇〇の理由で、今日限りです」
*   **★決断のフレーミング★**:
    「〇〇さん、ここまでの話を聞いて、今の自分のままでいいと思いますか？
    　それとも、本当はもっとやれるって、心のどこかで感じてますか？
    　……感じてるなら、あとは一歩踏み出すだけです」
*   **★沈黙★**: 相手が答えるまで黙る。穏やかな笑顔・アイコンタクト・微かなうなずき
*   **反応別対応**:

| 反応 | 対応 | テクニック |
|---|---|---|
| 「やります！」 | 「ありがとうございます！手続きに進みましょう」 | 即手続き。余計な話をしない |
| 「考えます」 | 「何が引っかかっていますか？」 | Feel-Felt-Found |
| 「お金がない」 | 「お金の問題がなかったら、やりたいですか？」 | 本当の反論を切り分け |
| 「相談してから」 | 「一緒に説明しましょうか？」 | 第三者を味方に |
| 「他と比較したい」 | 「比較するなら〇〇と〇〇を見てください」 | 比較基準をこちらが設定 |
| 「タイミングじゃない」 | 「いつならベスト？…実はそれ、一生来ない」 | 先延ばしの危険 |

*   **それでもNOの場合**:
    *   **ダウンセル**: 「まずは〇〇だけでも始めませんか？」
    *   **次回設定**: 「〇ヶ月後にまたお話しませんか？」
    *   **紹介依頼**: 「周りに困っている方がいたらご紹介ください」
    *   **★絶対ルール★**: いい関係で終わる。断られても笑顔で
