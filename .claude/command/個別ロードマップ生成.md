# 個別ロードマップ生成ワークフロー

クライアントの**過去の事実（職歴・実績・スキル）**と**目標・NG条件**をヒアリングし、「コンサルティング_ロードマップ完全ガイド」を元に個別最適化されたロードマップを自動生成します。

## 使用方法

1. Phase 1: ヒアリング（事実ベースの情報収集）
2. Phase 2: 現在地診断（6フェーズのどこにいるかを自動判定）
3. Phase 3: ロードマップ個別最適化（スキップ可能なタスク・追加タスクの特定）
4. Phase 4: 数字目標と期限の設定
5. Phase 5: アクションプラン出力（Markdownファイルとして保存）

# ★★★ AIがやりがちな失敗と対策 ★★★

## 失敗1: ターゲットや理想像を聞いてしまう

- **現象**: 「どんな人をターゲットにしたいですか？」「理想のペルソナは？」と聞く
- **対策**: ★ターゲットは定まっていない人が多い。聞くのは「過去の事実」のみ。コンセプトやターゲットはこちらが考えて提案する★

## 失敗2: 全員に同じロードマップを出す

- **現象**: Phase 1から順番にそのまま出すだけ
- **対策**: ★クライアントのスキル・経験に基づき、不要なステップはスキップ or 簡略化。重点的にやるべきフェーズを明示する★

## 失敗3: 曖昧なアクションプランを出す

- **現象**: 「コンセプトを考えましょう」のような抽象的な指示
- **対策**: ★「〇〇について△△の形式で□□までに書き出す」レベルまで具体化する★

## 失敗4: スキルを過大評価する

- **現象**: クライアントが「できます」と言ったスキルをそのまま信じる
- **対策**: ★具体的な実績・成果物で裏付けを確認する。「〇〇の経験はありますか？」ではなく「〇〇で〇〇を達成した経験はありますか？」と聞く★

## 失敗5: やりたくないことを無視する

- **現象**: 「顔出し嫌い」「通話営業できない」等のNG条件を考慮せず標準ロードマップを出す
- **対策**: ★NG条件に該当するタスクは代替手段を必ず提示する★

## Phase 2: 現在地診断

ヒアリング回答を基に、以下の診断ロジックでクライアントの**現在地（Phase）**を判定する。

### 判定ロジック

```
★以下の順番で上から判定する。最初にNoになった時点で、そのPhaseが現在地★

Q1. 自分の強み・実績・スキルを具体的な数字で言語化できているか？
    → No → Phase 1（土台構築）から

Q2. Xアカウントが存在し、プロフィール・コンセプトが明確に設計されているか？
    → No → Phase 1（土台構築）から

Q3. Xで毎日投稿し、フォロワーが100人以上いるか？
    → No → Phase 2（X運用開始）から

Q4. プレゼント企画を3回以上実施し、フォロワー500人以上・LINE100人以上いるか？
    → No → Phase 3（集客拡大）から

Q5. LINE→無料相談→成約の導線が1回でも通ったか？
    → No → Phase 4（マネタイズ導線構築）から

Q6. ウェビナーまたは有料noteで売上を出したか？セールストークが確立されているか？
    → No → Phase 5（セールス強化＆コンテンツ資産化）から

Q7. 自分の講座・スクールを持っているか？
    → No → Phase 6（講座化＆スケーリング）から
```

## Phase 3: ロードマップ個別最適化

「コンサルティング_ロードマップ完全ガイド.md」の各フェーズのタスクを、ヒアリング結果に基づいて個別最適化する。

### 最適化ルール

#### ルール1: スキップ判定
以下に該当するタスクは「✅ 完了済み」としてスキップする：
- すでに実施済みのタスク（例：自己分析が完了している、LINEが開設済み等）
- クライアントのスキルで不要なタスク（例：デザイン経験者はCanva学習不要等）

#### ルール2: 重点化判定
以下に該当するタスクは「⭐ 重点」としてマークし、詳細な手順を追加する：
- クライアントの弱点に関連するタスク
- 過去に失敗した経験に関連するタスク
- 目標達成に最もインパクトが大きいタスク

#### ルール3: NG条件の代替手段
NG条件に該当するタスクは、以下の代替手段テーブルを参照して差し替える：

| NGタスク | 代替手段 |
|---------|---------|
| 顔出し動画 | AIアバター動画 / テキスト＋画像投稿 / 音声のみ配信 |
| 音声配信 | テキストベースの長文投稿 / 画像コンテンツ（スライド化） |
| 通話営業 | テキストベースのセールス（LINE・DM） / オートウェビナー活用 / ステップ配信で自動化 |
| 毎日投稿 | 週3〜4投稿 + 予約投稿ツール活用 / n8nで自動化 |
| 長文ライティング | 音声→文字起こし→AI整形 / テンプレート活用 |
| 動画制作 | 画像スライド投稿 / テキスト長文投稿 / AIによるショート動画自動生成 |
| 対面コミュニケーション | チャットベースのサポート / 音声メッセージ |
| 広告費 | オーガニック集客に集中 / 引用・コミュニティ参加・コラボで露出 |

#### ルール4: スキルに応じたアドバイスのカスタマイズ

| クライアントの経験 | カスタマイズ内容 |
|------------------|----------------|
| ライティング経験あり | Phase 2-3の投稿クオリティを最初から高く設定。有料noteの早期リリースが可能 |
| デザイン経験あり | サムネ・スライド制作を自力で。ヘッダー画像・プレ企画の画像で差別化 |
| 営業経験あり | Phase 4のセールス部分を早期に開始。高成約率が期待できる |
| プログラミング経験あり | n8n自動化を早期に組み込み。ツール配布のプレゼント企画が作りやすい |
| AI利用経験豊富 | Phase全体でAI活用を標準装備。プロンプト配布のプレゼント企画がすぐにできる |
| 人に教えた経験あり | ウェビナー・セミナーへの移行が早い。Phase 5-6の到達が早まる |
| SNS運用経験あり | Phase 2を短縮可能。最初からPhase 3の施策を並行実施 |
| コンテンツ販売経験あり | Phase 4-5を大幅に短縮可能。ファネル設計に集中 |

### 出力フォーマット

各フェーズを以下のフォーマットで出力する：

```markdown
## Phase ○: [フェーズ名]

### あなたの場合のポイント
[このクライアントに特化したアドバイス。スキル・経験を踏まえた具体的な助言]

### やること一覧

| # | タスク | 状態 | 期限 | 詳細 |
|---|--------|------|------|------|
| 1 | [タスク名] | ✅完了済み / ⭐重点 / 通常 / 🔄代替 | ○週目 | [具体的にやること] |
| 2 | ... | ... | ... | ... |

### ⭐ 重点タスクの詳細手順
[重点マークがついたタスクについて、具体的なステップを書く]

### 🔄 代替手段の詳細
[NG条件で代替されたタスクについて、代替手段の具体的なやり方を書く]
```

## Phase 4: 数字目標と期限の設定

クライアントの目標月収・期限と、各フェーズの期間目安を照らし合わせ、個別の数字目標を設定する。

### 目標設定ルール

1. **逆算で期限を設定**：目標月収と期限から逆算して、各フェーズの完了期限を決める
2. **使える時間で現実調整**：1日に使える時間が少ない場合、各フェーズの期間を伸ばす
3. **スキルによる短縮**：経験のあるフェーズは期間を半分にする
4. **マイルストーンを設定**：各フェーズの完了を「小さなゴール」として設定し、達成感を得られるようにする

### 出力フォーマット

```markdown
## 📅 あなたのスケジュール

### 全体タイムライン

| フェーズ | 開始 | 完了予定 | 主な数字目標 |
|---------|------|---------|------------|
| Phase ○ | ○月○日 | ○月○日 | [具体的な数値] |
| ... | ... | ... | ... |

### 月別のやること

#### ○月（Phase ○）
- Week 1: [タスク]
- Week 2: [タスク]
- Week 3: [タスク]
- Week 4: [タスク]
- **月末チェック**: [達成すべき数字]
```

# 最終チェック

出力前に以下を自問してください。

- **ヒアリングは全て「過去の事実」に基づいているか？**: 「誰をターゲットにしたいですか？」等の未決事項を聞いていないか？
- **現在地の判定は正確か？**: スキルや経験を過大評価・過小評価していないか？
- **NG条件に代替手段を提示したか？**: やりたくないことを無視して標準ロードマップを出していないか？
- **スキップすべきタスクを特定したか？**: すでにできていることを繰り返し指示していないか？
- **重点タスクに具体的な手順を書いたか？**: 「○○を考えましょう」のような曖昧な指示になっていないか？
- **数字目標は現実的か？**: 使える時間・スキルレベルに対して無理な目標を設定していないか？
- **最初の1週間のタスクは超具体的か？**: 「今日から何をすればいいか」が明確か？
- **クライアントの強みを活かしているか？**: 経験やスキルを無視した汎用的なプランになっていないか？
