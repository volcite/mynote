# Phase 1: 🏗️ 土台構築

## 概要
**目的：自分が何者で、誰に何を届けるのかを明確にする**

すべての起点。ここが曖昧なまま発信を始めると、コンセプトがぶれて遠回りになる。自己分析と市場調査を徹底的に行い、「誰の」「何を」「どうやって解決するか」を言語化する。

## やること一覧

### ① 自己分析（最優先・1週間以内）
- **自分の持ち物・リソース・実績**を棚卸しする
  - スキル（例：AI、ライティング、デザイン、プログラミング、マーケ等）
  - 実績（例：フリーランス○年目、累計売上○万円、○○達成等）
  - AI活用の具体的成果（例：作業時間15時間→7時間に54%削減）
- 自己分析シートを使って漏れなく記入する

> [!TIP]
> **伝え方のコツ**: 「埋まらない箇所があっても大丈夫。まず全部書き出して、そこからブラッシュアップしましょう」と安心させる。完璧主義にならないように促す。

### ② 競合分析・市場調査（1週間以内）
- 同ジャンルの発信者を5〜10人リサーチ
- 以下を調べる：
  - どんなコンセプトで発信しているか
  - フォロワー数、投稿の傾向
  - マネタイズ方法（コンサル？note？講座？）
- 「唯一無二化」コンセプトシートを作成

### ③ コンセプト設計（最重要）
以下の4要素を明確にする：

| 要素 | 質問 | 例 |
|------|------|------|
| **誰の** | ターゲットは誰？ | AIに興味があるが使えていないフリーランスや会社員 |
| **何を** | どんな悩みを解決する？ | 労働時間が多い、AIで仕事奪われそう、使い方がわからない |
| **いつまでに** | フォローしてどのくらいで変われる？ | 3ヶ月後には |
| **どうやって** | 具体的な手段は？ | AI×n8nで業務自動化して労働時間を減らし売上を上げる |

> [!IMPORTANT]
> **コンセプトは途中で変わってもOK**。実際に発信しながら市場の反応を見て磨いていくもの。ただし最初の「仮決め」は絶対に必要。

### ④ プロフィール設計
- **名前**：覚えやすい4文字ひらがな + 肩書き（例：ごまさん｜AIで54%労働が短くなった犬）
- **肩書き**：何者かが一目でわかる + 実績 or 数字を入れる
- **プロフィール文**：
  - 得意なこと・スキル
  - 実績（数字で）
  - フォローするメリット
  - 所属コミュニティ（権威性）
- **アイコン**：AIで生成OK。シンプルな背景 + 人物 or 動物のイラスト
- **ヘッダー**：一目でアカウントの内容が伝わるデザイン

### ⑤ ツール・インフラ整備
- [ ] Xアカウント作成・プロフィール入力
- [ ] 公式LINE開設（プロライン推奨）
- [ ] 決済環境整備（Stripe → のちにUnivaPay）
- [ ] XTEP登録（プレゼント配布に使用）

# Phase 2: 🚀 X運用開始

## 概要
**目的：毎日の投稿習慣を身につけ、最初の100フォロワーを獲得する**

「発信の型」を覚え、量をこなすフェーズ。まだマネタイズは考えなくてよい。とにかく発信を止めないことが最重要。

## やること一覧

### ① 投稿の基本ルール

#### 投稿ネタの選び方
ペルソナが興味を持ちそうなことのみ呟く。それ以外は呟かない。

**ブックマークしたい投稿** を目指す：
- 後から見返そうと思えるもの
- パッと見で有益そうだと思えるもの
- 網羅的・体系的なもの
- みんなが知らなそうな情報
- 独り占めしたい、本当に教えたくない情報

#### 投稿の種類と配分

| 種類 | 割合 | 内容 | 目的 |
|------|------|------|------|
| **新規客向け** | 80〜90% | 有益情報、最新トレンド、ノウハウ | 認知拡大 |
| **既存客向け** | 10〜20% | マインド、思考法、ビジョン、業界への意見 | ファン化 |

#### ライティングの注意点
- **ひらがな：カタカナ：漢字 ＝ 5：3：2** のバランス
- カタカナ多すぎ・漢字多すぎは避ける
- 句読点を統一する
- AIに書かせてOK（特に意見系投稿）

### ② 露出を増やすアクション（超基本）

| アクション | 対象 | 目的 |
|-----------|------|------|
| **いいね** | 見込み客 | フォローしてもらう、商品買ってもらう |
| **いいね** | 同業者 | 彼らのフォロワーから認知を獲得 |
| **リプ** | 同業者 | Xのアルゴリズム評価UP → おすすめ表示 |
| **引用リポスト** | 同業者 | 彼らのフォロワーの目に入る |

#### 引用リポストの3条件
引用する相手は以下を満たす人：
1. 発信ジャンルが似ている（AI、マーケ、ライティング等）
2. フォロワー1000人〜3000人前後（大きすぎると反応もらいにくい）
3. 引用で絡んでくれた人に反応を返す人

### ③ 画像・動画の活用
- **機能的価値のある投稿**（有益情報、ノウハウ系）→ 画像 or 動画を付ける
- **体験談投稿**（イベント参加、勉強会等）→ 実際の写真・動画を付ける
- プレゼント配布時は4つ画像を載せる
- AIが動いている動画、2倍速の操作動画
- Soraなどでアイコンが動く動画を作るのもアリ

### ④ 自分の意見を言う投稿を増やす
- AI界隈に思っていること
- フリーランス界隈に思っていること
- 業界に対する持論、逆張り

> [!TIP]
> **AIで投稿を作ってOK**。ただし自分の意見・体験を核にすること。100%AIまかせだと刺さらない。

## Phase 2 の数字目標

- 平均インプレッション：200以上
- 毎日投稿を維持

## Phase 2 完了チェック

# Phase 3: 📈 集客拡大

## 概要
**目的：プレゼント企画を軸にフォロワー・LINEリスト数を一気に伸ばす**

フォロワー100人を超えたら、ここからが本番。「プレゼント企画」がフォロワー爆増の最大のレバレッジ。

## やること一覧

### ① プレゼント企画（集客の核）

#### 基本方針
**伸びている企画をパクる（＝参考にする）**

#### 企画ネタの種類

| 種類 | 具体例 | 用意の手軽さ |
|------|--------|------------|
| **プロンプト配布** | 「仕事で使えるプロンプト○選」 | ★★★ 超簡単 |
| **GPTs配布** | プロンプトのついでにGPTs化 | ★★☆ やや手間 |
| **ツール配布** | スプレッドシート、チェックリスト | ★★☆ |
| **音声配布** | スタエフで5分前後収録 | ★★★ 超簡単 |
| **レポート系** | 動画解説のPDF版 | ★★☆ |
| **長尺動画** | 30〜60分のYouTube動画 | ★☆☆ 手間だが価値高い |

#### 企画成功のポイント
- **トレンドに乗る**（新しいAIモデル発表時、話題のツール等）
- **口コミを必ずもらう**設計にする
  - メイン資料 → 無料
  - 口コミ特典 → 口コミ書いた人だけがもらえる追加コンテンツ
- **他ジャンル×AI** のネタで新規層にリーチ
  - アフィ界隈×AI、デザイン界隈×AI、資産運用界隈×AI 等
- **期間限定にして滑り込みで口コミ爆増**させる
  - 12時間前、6時間前、3時間前、1時間前に告知

#### プレゼント企画の頻度
- 最低でも2週間に1回
- プロンプト配布は週1でもOK（用意が楽）
- 3回に1回は変化球を入れる（音声、ツール、レポート等）

#### 告知投稿のネタ（プレゼント以外の長文投稿）
文字数500〜1000文字の長文投稿も強力：
- 「プロンプトの裏技的な書き方見つけました」
- 「Claudeでライティングさせるなら〇〇をしておけ」
- 「生成AIで売上2倍にしたい時、僕ならこうする」
- 海外で流行ってるのを使ってみたシリーズ

**構成**：
1. キャッチフレーズ（訴求性高く）
2. 有益情報（必ず学びがある）
3. CTA（〜してみて、報告して、ChatGPTに入力してみて）

### ② 他の集客施策

#### 質問受付イベント
- 「今から24時間以内にリプくれた人の質問全部答えます」
- リプ来た人に引用リポストで質問回答
- 1人目が来れば2人目も来る。1人目がポイント（サクラもアリ）
- 「24時間後に消します」と限定性を出す

#### 音声配信（スペース / スタエフ）
- 「今から1時間作業しながらスペース開くので質問ある人は来て！」
- テーマ例：プロンプトの書き方のコツ、AIで案件こなす時の意識
- 無料相談に繋げるオファーを毎回入れる

#### XTEPでのプチプレゼント
- 「こんなツール見つけたんだけど、知りたい人いる？」
- 「このプロンプトめちゃくちゃ便利なんだけど、欲しい人いる？」
- 途中までX上で見せて、途中からLINE登録で読めるようにする

#### 他の人のコミュニティに参加する
- オプチャ（オープンチャット）への参加
- 他のインフルエンサーのサロンでの活動

#### インフルエンサーへのDM拡散依頼
- 他ジャンルのプチインフルエンサーに引用してもらう

### ③ 公式LINE配信の開始
- **毎日配信（最低でも3日に1回）**
  - 理由：登録者の熱が冷めないようにする
- 配信内容：
  - X投稿に関連する内容 + プラスアルファ
  - 最後に「今日の投稿はこちらから」→XのURLを貼る

> [!WARNING]
> **集客がほぼOKになったら、それ以上時間をかけない**。次フェーズ（マネタイズ導線）に移行すべき。集客だけやっても売上にはならない。

# Phase 4: 💰 マネタイズ導線構築

## 概要
**目的：「X→LINE→無料相談→成約」の一連の流れを完成させる**

集客できても、売上に繋がる導線がなければ意味がない。このフェーズで「SNSを更新しているだけで相談者が増える仕組み」を作る。

## やること一覧

### ① ファネル設計（最重要）

```
X投稿 → フォロー
    ↓
固定ポストで無料プレゼント → LINE登録
    ↓
ステップ配信（教育）→ 信頼構築
    ↓
無料相談オファー → 申し込み
    ↓
事前アンケート → 日程調整 → 着座
    ↓
セールス → 成約
```

### ② LINE登録直後の設計

#### ステップ配信スケジュール
| タイミング | 内容 | 目的 |
|-----------|------|------|
| 登録直後 | あいさつ + プレゼント配布 | 期待値を高める |
| 5分後 | 無料相談案内 | 熱いうちにオファー |
| 30分後 | 無料相談案内（別角度） | 取りこぼし防止 |
| 1日後 8時 | 価値提供コンテンツ | 信頼構築 |
| 3日後 18時 | セールス | 行動促進 |
| 3日後 23時 | 締切訴求 | 緊急性 |

#### LINEで大切なこと
- 登録者の熱が冷めないうちに行動させる
- 配信では**双方向コミュニケーション**を取る
  - 音声配信、質問会、個人宛の音声メッセージ
- 口コミのスクショ画像を必ずオファー時に添付
- 実績を伝える
- 相談受けた後の未来を見せる
- 相談受けなかった時の未来を煽る

### ③ 無料相談の着座率を上げる方法

#### 申し込み〜着座までのチェックリスト

| 項目 | チェック |
|------|---------|
| LINEの1文目でオファーを認知できるか？ | □ |
| 登録から時間が経ちすぎていないか？ | □ |
| 期間限定にしているか？ | □ |
| 特典を渡しているか？ | □ |
| 教育（音声・質問会）できているか？ | □ |
| 相談で話す内容が伝わっているか？ | □ |
| 口コミを提示できているか？ | □ |
| 実績を伝えられているか？ | □ |
| 相談後の理想の未来が見えるか？ | □ |
| 相談しなかった時の最悪の未来を伝えているか？ | □ |
| 申し込み方法が明確か？（スマホ苦手な人でもOKか） | □ |
| 日程調整のラリーが多くないか？ | □ |
| 日程が先すぎないか？ | □ |
| Googleカレンダー連携で予約を整えているか？ | □ |
| リマインドを送っているか？ | □ |

#### 事前アンケート（必須4項目のみ）
1. 現状
2. 今の悩み
3. 欲しい未来
4. SNSアカウント

> Googleフォームで作成。項目は少なく、関係ないことは聞かない。

### ④ コンサル商品の設計

#### 松竹梅プラン（初期の価格例）

| プラン | 期間 | 料金 | 月額換算 |
|--------|------|------|---------|
| 梅 | 3ヶ月 | 9万円 | 月3万円（＝1日1000円） |
| 竹 | 半年 | 15万円 | 月2.5万円 |
| 松 | 1年 | 24万円 | 月2万円 |

#### モニター制度
- 最初はモニター価格で安く始める（例：3ヶ月5万円、月2万円等）
- **必ず期限を切る**（モニター価格終了のタイミングで通常価格へ）
- モニター期間中に感想・口コミを集める

#### 事前準備（準備が9割！）
- 相談者の情報からAIにロードマップを描かせておく
- 1個以上のアイデアを事前に用意
- 決済方法を確認（銀行振込 or Stripe）

> [!IMPORTANT]
> **地雷客の特徴 → 断ってOK**
> - 他社の講座を途中で辞めている・返金させている
> - 生活資金に余裕がなく来月生活できない
> - 直感的に「この人はヤバい」と感じる → 直感は信じていい

### ⑤ スクリーニング（客層管理）
**訴求の言葉選びで客層は変わる：**

| ❌ 悪い訴求（変な客が来る） | ✅ 良い訴求（本気の客が来る） |
|----------------------------|----------------------------|
| 一発逆転 | より売上を伸ばす |
| 誰でも簡単に | 売上出てるけど時間を作りたい |
| 楽して稼げる | 月商100万達成したけどもっと伸ばしたい |
| 簡単に稼げます | 年商1,000万目指しませんか？ |

# Phase 5: 🎤 セールス強化＆コンテンツ資産化

## 概要
**目的：セールス力を磨き、コンテンツを資産として蓄積し、売上を安定させる**

無料相談で成約率を上げつつ、ウェビナー・有料note・オートウェビナーなど「1対多」のセールス手法を確立する。

## やること一覧

### ① セールス（無料相談での営業フロー）

#### 通話セールスの流れ

```
1. 最初に「最後コンサル売るよ」と伝える
    ↓
2. 簡易的なロードマップを作る
    ↓
3. このロードマップ通りにやったら、どれだけ大変か（時間・お金・工数）を説明する
    ↓
4. 「この辺マルっと僕が全部サポートするので、○○さんは手を動かすだけだよ」
    ↓
5. やる？やらない？
    ↓
6. 「興味あります」→ 「そしたら資料あるんで説明しますね〜」
    ↓
7. 長々話すのではなく、相手の頭を働かせながら説明する
    ↓
8. 料金伝える前に「分割できるよ」「相場はこのくらいだよ」と伝える
    ↓
9. 料金伝えて、「どのサービスがいい？」と聞く
    ↓
10. 質疑応答タイム
```

#### セールスで意識すること

**ベネフィット（理想の未来）**
- 超具体的にイメージさせる
- 相手が本当に欲しいものか確認する
- 例：「毎日寝る前にセットした企画が伸びて、朝起きたらLINEの登録者が増えていて、通知がたくさん来てる状態で起きて、朝コーヒー入れてみんなのコメントに返す」

**ペイン（最悪の未来）**
- 超具体的にイメージさせる
- 「今すぐやらないといけない」理由をいろんな角度から伝えまくる
- ペルソナの心の声を30個書き出しておく

**感情を揺れ動かす**
- プラスの感情（希望、ワクワク）とマイナスの感情（不安、焦り）を交互に
- リアリティを持って相手にイメージさせる

#### やりがちな失敗と対策

| 失敗 | 対策 |
|------|------|
| コンサルしすぎて「自分でやります」と言われる | 不満を残す。全部教えない |
| オファーせずに終わる | 最初に宣伝予告する。「売り込みを怖がらない」マインド |
| 長くなりすぎて相談オファーの時間がない | 時間区切りをしっかり。リハーサル必須 |
| 特典の説明を忘れる | リハーサル |

#### オファーがしにくい時のマインド
- **オファーしないことがお客様のためではない**
- 宣伝予告を先に打つ（「最後にちょっとお知らせがあります」）
- 普段の配信で「コンサルで教えている」「コンサル募集しようかな」と匂わせる
- ChatGPTと壁打ちしてトーク練習

### ② ウェビナー（1対多セールスの武器）

#### ウェビナー実施の流れ
1. **構成作成** → 台本作成 → スライド作成 → サムネイル制作
2. **告知**：1週間前から。めちゃくちゃしつこいくらいが丁度良い
3. **リハーサル**：必ず1回やる（時間オーバー防止、特典忘れ防止）
4. **本番**：15〜20分程度で簡潔に
5. **無料相談オファー**：セミナー終了直後に限定オファー
   - 例：「セミナー参加者限定、10人限定、終了後72時間」

#### ウェビナー後の振り返りチェック
- [ ] セミナーを時間内に完結できたか
- [ ] 話はスムーズだったか
- [ ] 手を動かすデモはスムーズだったか
- [ ] 無料相談オファーをちゃんとできたか
- [ ] 特典の説明を忘れなかったか
- [ ] 告知は十分にしたか

#### オートウェビナー化（自動化）
- 一度撮影したウェビナーをオートウェビナーページに設置
- LINE登録 → オートウェビナー視聴 → 個別相談会 → 契約の導線
- プロラインで作成（参加者タグ付け含む）
- **それぞれの遷移率を追いかける**

個別コンサルだけでは時間の限界がある。動画講義・カリキュラム化によってレバレッジを効かせ、スケールさせる。

## やること一覧

### ① 講座のカリキュラム設計

**３つのルート（クライアントに合わせて提案）：**

#### ルート①：クライアントワーク
| STEP | 内容 |
|------|------|
| STEP 1（準備） | 自分の業務をn8nで自動化してポートフォリオ作成。CW等で小規模案件（数万〜10万）をこなし実績と声を集める |
| STEP 2（月収100万） | 信頼したクライアントに業務フロー全体の自動化（30万〜50万）を提案。保守運用の継続契約（月5万〜10万）を結ぶ |
| STEP 3（拡大/自由） | ディレクターとして案件管理に回り外注。または同業者向け育成講座を販売 |

#### ルート②：ツール・SaaS販売
| STEP | 内容 |
|------|------|
| STEP 1（準備） | 汎用的なn8nワークフローテンプレートを作成。販売〜納品の自動化システムを構築 |
| STEP 2（月収100万） | SNS・広告でアクセスを流してテンプレート販売。購入者に導入・カスタマイズ代行（30万〜50万）をアップセル |
| STEP 3（拡大/自由） | 独自SaaS（月額課金）を開発。またはテンプレート販売とアフィリのみに集約 |

| STEP | 内容 |

5. マニュアルを用意（無料配布・有料販売もOK）

**理想のチーム構成：** 自分 → メンバーA、メンバーB
- 片方が抜けてもチームとして機能 + 人数が多すぎないライン
