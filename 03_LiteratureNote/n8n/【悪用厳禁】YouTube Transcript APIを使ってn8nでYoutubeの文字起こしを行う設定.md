## YouTube Transcript APIの設定

YouTube Transcript APIを使ってn8nでYoutubeの文字起こしを実現します。

まずYouTube Transcript APIにアクセスします。

[**Youtube Transcript API** *Generate YouTube transcripts instantly with our free tool. Ex* *www.youtube-transcript.io*](https://www.youtube-transcript.io/api)

右上の「My Account」をクリックします。

![画像](https://assets.st-note.com/img/1764142852-nMsd6Oo2RhyqEX3eZj0IQ9Ni.png?width=1200)

ログインボタンをクリックして、ログインまたはアカウント作成を実施します。

![画像](https://assets.st-note.com/img/1764142892-l2IFD97LWkfYVUgx0v4wBMPo.png?width=1200)

![画像](https://assets.st-note.com/img/1764142904-l5kbwTgp13jqAMLuWSn2Is8N.png?width=1200)

ログインすると下記画面に遷移するので、「Generate API Token」をクリックします。

![画像](https://assets.st-note.com/img/1764144629-j4dcQK1HNnX83RplTqWU9suF.png?width=1200)

利用規約に同意する画面が出る可能性があるので、「Agree and Generate」をクリックしてAPIキーを生成しましょう。

![画像](https://assets.st-note.com/img/1764145241-pCAm54KXjbMGiQEyc8Blq3gY.png?width=1200)

生成したAPIキーをコピーしてメモ帳などに保存しておきます。

![画像](https://assets.st-note.com/img/1764145246-wNCkqUrx2geLhGTd6E1HaVyD.png?width=1200)

## Summary
n8nとYouTube Transcript APIを組み合わせることで、YouTube動画の文字起こしを自動化し、情報を効率的に収集する方法について解説された記事です。
Pythonなどのコードを書くことなく、APIキー（Credentials）を設定するだけで、動画データの文字起こしや要約が可能になります。具体的な手順として、YouTube Transcript API公式サイトでのアカウント作成、ログイン、APIトークンの生成、そしてAPIキーの保存までの流れが画像付きで説明されています。
この自動化により、競合動画の分析やコンテンツ作成のネタ収集をAIに任せ、自身の時間を創造的な活動に充てることができるようになります。

## Key Learning Points
*   **YouTube Transcript APIの活用**: 複雑なスクレイピングやプログラミングを行わずに、YouTubeの文字起こしデータを取得する手段として有効です。
*   **APIキーの取得手順**:
    1.  YouTube Transcript API公式サイトへアクセスし、「My Account」からログインまたはアカウント作成を行う。
    2.  「Generate API Token」をクリックする。
    3.  利用規約に同意（Agree and Generate）し、表示されたAPIキーをコピーして保存する。
*   **n8nとの連携**: 取得したAPIキーをn8nで設定することで、YouTube上のデータを自動的に収集・整理するワークフローが構築可能です。
