## Summary
ノーコードツール「n8n」を使用し、スピリチュアル向けのInstagramショート動画（リール）を全自動で生成・投稿する仕組みを解説した記事。
顔出し・声出し・撮影なしで再生数を稼ぐため、ChatGPTによるネタ作成、Geminiによる画像生成、Runwayによる動画化、そしてFacebook Graph APIを用いたInstagramへの自動投稿までを連携させたワークフローを紹介。
編集時間を0秒にし、「供給の蛇口をひねりっぱなしにする」ことで、寝ている間も自動で運用し影響力を拡大する手法を提案している。

## Key Learning Points
- **ワークフローに必要な事前準備**
    - n8nアカウント
    - 各種APIクレデンシャル（OpenAI API, Google Cloud Storage, Runway API, Facebook Graph API）
- **自動化ワークフローの3ステップ詳細**
    1.  **AIによるネタ作成**:
        - `ScheduleTrigger`で毎日決まった時間に実行。
        - `AI Agent` (ChatGPT 4o) がスピリチュアル動画用のタイトル・ランキング・キャプションをJSON形式で生成。
        - `Gemini` (models/gemini-3-pro-image-preview) でタイトル入りの画像を生成。
        - `Google Cloud Storage`へ画像を保存し、公開URLを取得。
    2.  **動画生成**:
        - `Runway API`の`image_to_video` (モデル: veo3.1) を使用し、静止画の一部のみを動かすプロンプトで動画化。
        - 生成完了まで待機し、生成された動画ファイルを取得。
    3.  **Instagramへの自動投稿**:
        - 動画を再度Cloud Storageに保存してURL化。
        - `Facebook Graph API`を使用し、Reelsとして動画コンテナを作成し、公開処理を実行。
- **n8n活用の利点**
    - 手作業での素材探しや編集作業を全廃できる。
    - 24時間稼働でコンテンツを供給し続け、フォロワー獲得を効率化できる。

## Key Message
手作業での動画制作は時間の無駄であり、n8nを活用して「供給の蛇口をひねりっぱなしにする」ことこそが重要である。テクノロジーを活用して労働から解放され、寝ている間に分身に働かせる仕組みを持つことで、人生の時間を有意義に使うべきだ。
