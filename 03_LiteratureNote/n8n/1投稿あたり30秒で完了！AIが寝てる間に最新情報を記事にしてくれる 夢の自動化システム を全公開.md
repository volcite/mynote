### ノード② RSS Feed Read（VentureBeatからニュースを取得）

**このノードの役割**：海外テックメディアから最新ニュースを自動で取ってくる

**RSSとは？**

RSS（Really Simple Syndication）は、Webサイトの更新情報を配信する仕組みです。

簡単に言うと「**自動で最新記事をお届けしてくれるサービス**」です。

**設定内容**：

```

```

このURLを指定するだけで、VentureBeatの最新記事が自動で取得されます。

### ノード③ Code（1週間以内の記事をフィルタ）

**このノードの役割**：古い記事を除外して、新しい記事だけを残す「ふるい」

RSSで取得した記事には、1ヶ月前や3ヶ月前の古い記事も含まれています。

でも私たちが欲しいのは「**先週の最新情報**」だけ。

このノードでは、過去7日間（1週間）以内に公開された記事だけを残します。

**何をしているか（簡単に）**：

1.  今日の日付を確認する
    
2.  7日前の日付を計算する
    
3.  それより新しい記事だけを残す
    
4.  古い記事は捨てる
    

[![画像](https://assets.st-note.com/img/1770656420-erot2TdWmXQaq07HG61bRjAh.png?width=1200)](https://assets.st-note.com/img/1770656420-erot2TdWmXQaq07HG61bRjAh.png?width=2000&height=2000&fit=bounds&quality=85)

プログラム

```
// 1週間（7日間）以内の記事のみをフィルタリング
const now = new Date();
const oneWeekAgo = new Date(now.getTime() - 7 * 24 * 60 * 60 * 1000);

const filteredItems = items.filter(item => {
  const pubDate = new Date(item.json.pubDate);
  return pubDate >= oneWeekAgo;
});

// 記事が0件の場合は空配列を返す
if (filteredItems.length === 0) {
  return [{ json: { noArticles: true, message: '過去1週間以内の新着記事はありませんでした。' } }];
}

return filteredItems;
```

もし1週間以内の新着記事が0件だった場合は、「記事がありませんでした」というメッセージを返します。
