## 第1章：【全貌公開】これが月100万を自動で引き寄せる「営業の仕組み」だ

まずは、今回構築する「全自動営業ハンターシステム」の全体像をお見せします。

この図を見てください。（脳内でイメージしてください）

1.  **【情報収集】** n8nが、 **クラウドワークス、ランサーズ、ココナラ** の3大プラットフォームを監視。
    
2.  **【選別】** あなたが設定した「NGワード（例：動画編集、モニター、低単価）」を除外し、「優良案件」だけを抽出。
    
3.  **【執筆】** 抽出された案件の募集文をAI（ **Gemini** ）が読み込み、そのクライアントに刺さる「専用の提案文」を執筆。
    
4.  **【通知】** あなたのスマホ（メール）に、「案件URL」と「生成された提案文」が毎朝届く。
    

あなたは、この通知を見て、  
「お、これいいな」  
と思ったら、提案文をコピーして、微調整（自分の名前を入れる程度）して、送信するだけ。

### なぜ「全自動応募」ではなく「コピペ（半自動）」なのか？

「どうせなら応募まで自動化してよ」

そう思った方もいるかもしれません。技術的には可能です。  
しかし、私はあえて **「最後の送信ボタン」だけは人間が押す** ことを強く推奨しています。

理由は2つあります。

**1\. アカウント停止リスクの回避**  
機械的な連続投稿は、プラットフォーム側からスパム認定されるリスクがあります。  
特にクラウドソーシングサイトは自動化botに厳しいため、IP分散などを考慮しない完全自動化は「自爆」行為です。

**2\. 「魂」の注入**  
AIは優秀ですが、たまに「的外れ」なことを言います。  
また、クライアントの募集文に「最後に『りんご』と書いてください」といった、自動応募避けのトラップがある場合もあります。

これらを人間が最後にチラッと確認するだけで、 **「採用率」は劇的に向上** します。  
完全に全てをAIに丸投げするのではなく、 **「面倒な99%」をAIにやらせて、「重要な1%」だけ人間がやる** 。

この「AIの圧倒的パワー」と「人間の最終チェック」のハイブリッドこそが、最強の営業スタイルなのです。

## 第4章：【構築編2】「採用率」を劇的に上げる悪魔的プロンプト

案件を厳選したら、次は「提案文」の作成です。  
ここが**このシステムの心臓部**であり、最も重要なパートです。

多くの人は、AIにこう指示してしまいます。  
「この案件の提案文を書いて」

**…これでは、0点です。**  
ありきたりなAI文章が出来上がり、クライアントに見抜かれて即ゴミ箱行きです。

月100万稼ぐためのプロンプトは、設計思想が違います。

### クライアントが「即採用」したくなる心理トリガー

クライアントが提案文を読む時間は、平均「3秒」です。  
この3秒で、「お、こいつは違う」と思わせるには、以下の3要素が必要です。

1.  **共感（あなたの課題を理解しています）**
    
2.  **権威性（私なら解決できます、なぜなら…）**
    
3.  **限定性（今ならこの特典がつきます）**
    

これをAIに強制的に出力させる、私の「秘伝のプロンプト」を公開します。

### 【コピペOK】全自動提案文生成プロンプト

n8nの**OpenAIノード**に、以下のプロンプトを設定してください。

```
# Role
あなたは、Web制作・システム開発におけるトップセールスマンです。
クライアントの潜在的な課題を見抜き、成約率の高い提案文を作成することが使命です。

# Input Data
案件タイトル: {{ $json.title }}
案件詳細: {{ $json.contentSnippet }}

# Your Information（ユーザーの実績）
- 経験年数: 5年
- 得意分野: 業務効率化、自動化ツール開発
- 実績: クラウドワークスにて100件以上の納品実績あり

# Instructions
以下の構成に従って、人間味のある、熱意のこもった提案文を作成してください。
AIっぽい堅苦しい表現は禁止です。「〜だと思います」ではなく「〜と確信しています」と言い切ってください。

## 構成案
1. **挨拶と共感**: 案件詳細から「クライアントが何に困っているか」を一言で言い当てる。
2. **解決策の提示**: 自分のスキルでどう解決できるか具体的に（技術用語は使いすぎない）。
3. **差別化提案**: 「さらに、今回私に任せていただければ、〇〇もサービスします」という特典をつける。
4. **アクション**: 「まずはZoomで15分ほどお話できませんか？」と締めくくる。

# Output
提案文の本文のみを出力してください。
```

このプロンプトを使うと、AIは単なる「応募」ではなく、\*\*「クライアントへのラブレター」\*\*を書き始めます。  
これが、自動化なのに人間味が出る秘密です。
