# 🔐 【ご購入者限定】X運用完全自動化システムの構築手順

ご購入ありがとうございます！
ここからは、実際に「ClaudeCode × n8n X運用完全自動化システム」をあなたの手元に構築していく手順を解説します。

※本番のフォルダとスクリプト群は、この記事の最下部にあるダウンロードリンクから取得してください。

## Step 1: 必要な環境の準備

まずは大前提となるツールを揃えましょう。

- bash 環境 (Linux, macOS, Windows WSLなど)
- Claude Code (CLIツール)のインストール
- Anthropic APIキーの取得
- n8nのアカウント（自動投稿用のWebhook受信に使用）

## Step 2: ワンクリック自動セットアップ

ダウンロードした `x-automation` フォルダを開いてください。
難しい初期設定は、AIがすべて自動でやってくれます。

ターミナル（黒い画面）を開き、以下のコマンドを実行します。

```bash
# 引数として "n8n Webhook URL" と "運用するXアカウントID" を渡します
bash scripts/00_run_setup.sh "https://your-webhook-url-here..." "@your_x_account"
```

これを実行するだけで、AIがあなたのアカウントを分析し、最適なペルソナ、文体（トーン＆マナー）、投稿戦略を自動で生成し `data/` フォルダに保存してくれます。

## Step 3: 手動での微調整（よりあなたらしく）

AIが作った初期設定を、少しだけあなた好みにチューニングしましょう。
`data` フォルダの中にある以下のファイルを確認してください。

1. **`persona.md`**: あなたの強みや競合情報がまとまっています。違和感があれば修正してください。
2. **`style_guide.md`**: 文章のクセや語尾の設定です。
3. **`strategy.md`**: 投稿時間やアカウント運用の方針です。

## Step 4: モーニングパイプラインの稼働

いよいよ自動化の実行です。

```bash
cd scripts/
./pipeline_morning.sh
```

このスクリプトを走らせるだけで、以下のフローが全自動で進みます。

1. **Research**: トレンド分析・競合調査
2. **Planning**: 投稿プラン作成
3. **Writing**: 下書き（ドラフト）生成
4. **Editing**: 品質チェックと承認
5. **Webhook**: 完成した投稿をn8nに送信 → 自動投稿！

これを毎朝の手動実行にするのも良し、n8nのCronノードやOSのタスクスケジューラなどで「毎朝7時に自動実行」と設定すれば、**文字通りの「完全放置」運用**がスタートします。
